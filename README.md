# 图像处理入门

## 1.图像读取
实现对常见图片格式的读取、显示、存储操作。
## 2.直方图
1. 计算并显示图像灰度直方图，并进行直方图均衡化。  
2. 实现彩色图像与灰度图像的转化，彩色图像通道提取。  
3. 彩色图像直方图显示，彩色图像直方图均衡。
## 3.进度条控件  
熟悉进度条控件引入的回调函数机制。  
## 4.图像分割
1. 实现图像阈值分割，查找并绘制图片轮廓。  
2. 实现基础的区域生长算法。  
## 5.形态学变换
1. 实现图像形态学变化中的膨胀、腐蚀。  
2. 实现图像形态学变化中的开运算、闭运算。
